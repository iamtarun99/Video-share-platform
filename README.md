## Description

Video Share and social media platform: A NestJS Monorepo project with Gateway and Microservices

## Installation

```bash
$ npm install
```

## Set up .env file
  -Set the credentials and parameters for given variables in .env file
  
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
